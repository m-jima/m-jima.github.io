Basic Fusion Rules

Normal Fusion

Formula

  1. Ingredient A x Ingredient B = ?
    • Consult the game's corresponding demon list to obtain each ingredient's race and base level.
  2. Determine Race 'R'
    • Consult the game's normal fusion chart.
    • Find the row that starts or ends with Race 'A'
    • Find the column that starts or ends with Race 'B'
    • Race 'R' = Intersection of Race 'A' and Race 'B'
    • If Race 'R' ISNA, the fusion is invalid.
    • (Persona only) Insersection must lie in upper-right half of race chart. If it does not, flip Race A and Race B.
  3. Determine Result R
    • Consult the game's demon list
    • Level R = (Base Level A + Base Level B + 1) >> Game-specific modifier
    • Result R = Next-highest non-unique result with Race R and base level >= Level R
    • If Level R > All non-unique results with Race R, the fusion is invalid.
  4. Ingredient A x Ingredient B = Result R

Example 1

  1. Shin Megami Tensei IV: Current Level 53 Osiris x Current Level 26 Principality = ?
    • List of Demons - Shin Megami Tensei IV
    • Osiris: Race A = Deity, Base Level A = 53
    • Principality: Race B = Divine, Base Level B = 26
  2. Determine Race R
    • Fusion Chart - Shin Megami Tensei IV
    • Deity (row) x Divine (column) = Herald
  3. Determine Result R
    • SMT IV Normal Modifier = 1
    • Level R = (53 + 26 + 1) >> 1 = 40
    • Next-highest Herald Demon with base level >= 40 is Israfel (45), not Victor (40)
  4. Shin Megami Tensei IV: Current Level 53 Osiris x Current Level 26 Principality = Base Level 45 Israfel

Triple Fusion

Formula

  1. Ingredient A x B x C = ?
    • Consult game's demon list for race and base level
    • Ingredient C = Demon with MAX current level
    • If multiple ingredients are tied for MAX current, Ingredient C = Ingredient with highest race priority
    • By default, demons are sorted from higher to lower race priorities (consult demon list for game)
  2. Determine Race AB
    • Consult normal fusion chart
    • Find the row which starts/ends with Race A
    • Find the column which starts/ends with Race B
    • Race AB = Intersection of Race A x Race B
    • If Race R = None, the fusion is invalid.
  3. Determine Race R
    • Consult triple fusion chart
    • Find row which starts/ends with Race AB
    • Find column which starts/ends with Race C
    • Race R = Intersection of Race AB x Race C
    • If Race R = None, the fusion is invalid.
  4. Determine Result R
    • Consult demon list
    • Level R = Math.floor((Level A + Level B + Level C) / 3) + Triple Modifier
    • Result R = Next-highest non-unique result with Race R and base level >= Level R
    • If Level R > All non-unique results with Race R, the fusion is invalid
  5. Ingredient A x Ingredient B x Ingredient C = Result R
